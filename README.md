FantasyMap
==========
This is a mapping tool build primarily for Fantasy games. 
It is designed for someoen to make a map of whatever they 
choose. It has the capability of having multiple clients 
and one mast client. 

It is built with:
-html5 canvas
-node.js

Run:
go to the root directory and run 

''node server/server.js"

then access "ip.ip.ip.ip:8000/map.html"
